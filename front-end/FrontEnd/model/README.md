# Model Assets

AI model files for on-device inference:

- `yolov8n.mnn` — YOLOv8n (MNN format) for scene object detection
- `movenet_lightning.tflite` — MoveNet Lightning for real-time pose keypoint detection
- `blazepose.tflite` — BlazePose for offline template keypoint extraction
- `nima_mobilenet.mnn` — NIMA-MobileNet (MNN format) for aesthetic scoring (reserved)

These files are large and not committed to the main repository.
Use Git LFS or a download script to acquire them.
